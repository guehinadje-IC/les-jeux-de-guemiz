# 🌴 Les Jeux de Guémiz

**Le jeu de l'actu en famille** — Guadeloupe & le monde. Quiz, débats d'idées et culture générale pour ados et adultes (11-99 ans). Du lundi au samedi, avec un défi bonus le samedi.

👉 **Jouer :** _(le lien GitHub Pages s'affichera ici après publication)_

## Ce que contient le jeu
- 🏝️ **Lundi** — présentation journalistique sur une actu locale (Guadeloupe)
- 🌍 **Mardi** — actu internationale
- 💬 **Mercredi** — un grand débat d'idées (arguments pour / contre)
- 🧠 **Jeudi** — quiz sur l'actu de la semaine
- 📚 **Vendredi** — quiz culture générale (utile à l'école)
- 🏆 **Samedi** — Défi Famille bonus (récap points ×2 + jeux)

Système de points, rangs et séries pour jouer toute la semaine.

## Comment ça marche
Tout tient dans un seul fichier : **`index.html`**. Aucune installation, aucun serveur. Il suffit de l'ouvrir dans un navigateur, ou de le publier sur **GitHub Pages** pour le partager par un simple lien.
